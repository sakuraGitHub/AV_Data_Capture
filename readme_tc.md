**日本電影元數據 抓取工具 | 刮削器**，配合本地影片管理軟體 Emby, Jellyfin, Kodi 等管理本地影片，該軟體起到分類與元數據（metadata）抓取作用，利用元數據信息來分類，僅供本地影片分類整理使用。
#### 本地電影刮削與整理一體化解決方案
